# ios7crypt - Encrypt and decrypt Cisco IOS7 passwords

# EXAMPLE

    $ npm install
    $ make test
    node cli.js -e monkey
    00091c080f5e12
    node cli.js -d 00091c080f5e12
    monkey

# HOMEPAGE

http://ios7.yellosoft.us/

# REQUIREMENTS

* [Node.js](http://nodejs.org/) 0.10+

# INSTALL

    $ npm install ios7crypt

# LINT

Keep the code tidy:

    $ npm run-script lint
